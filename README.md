# claps-health-mobile
This repository contain the code, assets, and resources for the Claps Health mobile app.


## Prerequisites

Before you can run this application, you will need the following:

- Flutter version: `3.7.12`


## Getting Started

To start the application, follow these steps:

1. Run `flutter pub get` to install the dependencies.
